Para recriar: usar grid CSS com o texto do hero alinhado à esquerda, pill badge acima do H1 em vermelho com borda radius 100px, H1 em Barlow Condensed 8vw. A seção de serviços deve usar CSS Grid com grid-template-columns: 60px 1fr 40% — número grande na primeira coluna, texto no meio, imagem do serviço na terceira. As animações de entrada devem ser leves e suaves (nada de bounce ou elastic), sempre com easing cubic-bezier elegante. O diferencial deste template para a Magrão é a forma como a numeração organiza visualmente uma lista longa de serviços — perfeito para os +25 serviços do portfólio.

7 — SISTEMA DE VARIAÇÃO DE LAYOUT
Escolhas registradas — justificativa baseada no nicho, mídias disponíveis e branding:
HERO — [X] D) Vídeo loop + texto com máscara de cor
→ Justificativa: 24 vídeos disponíveis sobre ambiente, produtos e equipamentos da oficina. Usar vídeo real no hero é o diferencial máximo de autenticidade. Overlay vermelho diagonal esquerdo cria máscara de cor da identidade sem esconder o vídeo.
SERVIÇOS — [X] F) Numeração grande (01, 02, 03) como elemento visual
→ Justificativa: Com +25 serviços no portfólio, a numeração organiza sem sobrecarregar. Os números em 8rem vermelho semitransparente criam hierarquia visual e remetem à precisão técnica/mecânica.
DEPOIMENTOS — [X] A) Marquee infinito horizontal
→ Justificativa: 327 avaliações no Google com 4.4 estrelas é um ativo enorme de credibilidade. O marquee permite mostrar múltiplos depoimentos de forma contínua e energética, combinando com o logo oficial do Google para chancela.
SOBRE / CREDENCIAIS — [X] D) Split 50/50 com imagem fixada
→ Justificativa: A história de Cláudia é o coração emocional da marca. Foto da equipe fixada à esquerda + narrativa + 4 counters animados (19 anos / 327 avaliações / +25 serviços / Atendimento feminino) à direita criam o equilíbrio perfeito entre emoção e credibilidade técnica.
